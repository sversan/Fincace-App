Despre creatorul limbajului Python 🐍

Guido van Rossum este creatorul limbajului de programare Python.

A început să lucreze la Python la sfârșitul anilor 1980, iar prima versiune publică a apărut în 1991.

Scopul lui a fost să creeze un limbaj de programare ușor de învățat, dar puternic, care să poată fi folosit pentru proiecte mari și mici.

Numele "Python" provine din trupa de comedie britanică "Monty Python", și nu de la șarpe.
